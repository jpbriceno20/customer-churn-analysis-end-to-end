# Data Notes

This project includes a raw customer-level dataset used to perform churn analysis.

## Raw Data
The file located in `data/raw/` represents structured customer account data commonly found in subscription-based businesses.

The dataset includes attributes related to:
- Customer profile and demographics
- Contract and tenure information
- Service usage indicators
- Billing, charges, and payment methods

## Data Usage
The raw dataset serves as the input for:
- SQL-based data exploration and cleaning
- Analytical view creation
- Machine learning model training and prediction

All transformations and preparation steps are documented in the SQL scripts included in this repository.

## Privacy
The dataset is publicly available, simulated, or anonymized.
No real customer-identifiable or proprietary company data is included.
